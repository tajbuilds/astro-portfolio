# Security Policy

## Supported Scope

This repository contains portfolio code and public case-study content.

- Runtime platform: Cloudflare Workers + static assets
- Contact endpoint: `/api/contact`
- Anti-abuse: Cloudflare Turnstile + server-side verification

## Reporting a Vulnerability

Please report security issues privately by email:

- contact@tajs.io

Include:

- A clear description of the issue
- Reproduction steps
- Expected vs actual behavior
- Impact assessment

Do not open public GitHub issues for security vulnerabilities.

## Disclosure Expectations

- I will acknowledge valid reports as quickly as possible.
- I may request additional details for reproduction/validation.
- Sensitive infrastructure details should not be disclosed publicly.

## Notes

Case-study content in this repo intentionally uses generalized/sanitized identifiers.
No private endpoints, secrets, or customer-sensitive routes should be published.
